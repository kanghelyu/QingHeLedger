# Qinghe Ledger · 青禾记账

**中文** — 本地优先的多语言记账 App，支持计算器记账、账单编辑、统计、惯用货币和 Frankfurter 日更汇率。账单只保存在设备上。

**English** — A local-first multilingual expense tracker with calculator entry, editing, analytics, preferred currency, and daily Frankfurter rates. Entries stay on your device.

**हिन्दी** — स्थानीय-प्रथम बहुभाषी खर्च ऐप, जिसमें कैलकुलेटर, संपादन, आँकड़े, पसंदीदा मुद्रा और दैनिक Frankfurter दरें हैं। डेटा आपके डिवाइस पर रहता है।

**Español** — Gestor de gastos multilingüe y local, con calculadora, edición, estadísticas, moneda preferida y tipos diarios de Frankfurter. Los datos permanecen en el dispositivo.

**العربية** — تطبيق مصروفات متعدد اللغات يعمل محليًا، مع آلة حاسبة وتعديل وإحصاءات وعملة مفضلة وأسعار Frankfurter اليومية. تبقى البيانات على جهازك.

**Français** — Gestionnaire de dépenses multilingue et local, avec calculatrice, modification, statistiques, devise préférée et taux Frankfurter quotidiens. Les données restent sur l’appareil.

**Русский** — Локальное многоязычное приложение для расходов с калькулятором, редактированием, статистикой, основной валютой и ежедневными курсами Frankfurter. Данные остаются на устройстве.

**Português** — Controle de despesas multilíngue e local, com calculadora, edição, estatísticas, moeda preferida e taxas diárias Frankfurter. Os dados ficam no dispositivo.

## Build

Open the generated folder in Android Studio, use JDK 17, sync Gradle, and run on Android 8.0 or newer.
